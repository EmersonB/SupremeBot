# Supreme-Bot

# Instructions
1. Open Terminal
2. type "cd downloads/supremebot"
3. If first time running
  2.1 chmod +x setup.sh
  2.2 sh setup.sh
4. If already setup
  3.1 chmod +x start.sh
  3.2 sh start.sh

#Adding chrome extension
1. Locate chrome extensions here - chrome://extensions/
2. Enable developer mode
3. Press load unpacked extension and navigate to “Supreme-Bot-Chrome-Extension” and select folder
4. Right click the extension and press options
